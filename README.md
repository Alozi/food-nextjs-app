# Food Next.js App

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

This is a Food Next.js App web application built with React and Next.js. The goal of this project practice with Next.js.

![Screenshot with game](/public/screencapture-1.png)

![Screenshot with game](/public/screencapture-2.png)

## Run Project

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
